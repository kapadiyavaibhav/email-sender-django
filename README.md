# Email Sender Automation System

## Description
A Django-based web application used to send automated emails using SMTP integration.

## Features
- Email sending functionality
- User authentication
- SMTP configuration
- Database integration
- Backend development using Django

## Technologies Used
- Python
- Django
- MySQL
- HTML
- CSS

## Project Structure
- email_app/
- email_sender/
- manage.py
